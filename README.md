# SE-Wallet
Stock Exchange Wallet: stocks, investment funds

# Test programme:

1. Install [node.js](https://nodejs.org/en/) with [npm](https://www.npmjs.com/)
2. Install [nodemon](https://www.npmjs.com/package/nodemon) via npm

    ```
    npm install -g nodemon
    ```
3. Run nodemon

    ```
    nodemon -q main.js
    ```
